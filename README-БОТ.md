# 🤖 Авто-напоминания в Telegram (каждый день в 09:00)

Это «будильник в облаке» на **GitHub Actions** — бесплатно, работает 24/7, твой компьютер включать не нужно.
Каждый день в 09:00 МСК бот сам присылает дайджест оплат в ваш чат.

---

## Что нужно один раз настроить (≈15 минут)

### 1. Перевыпусти токен бота
Старый токен ты присылала открыто — его надо заменить.
- Открой **@BotFather** в Telegram → команда `/revoke` → выбери `payment_ddma_bot` → получишь **новый токен**. Скопируй.

### 2. Заведи репозиторий на GitHub и загрузи туда папку `paytracker`
- Зарегистрируйся на https://github.com (если ещё нет).
- Нажми **New repository** → назови `paytracker` → поставь **Private** → Create.
- На странице репозитория: **Add file → Upload files** → перетащи туда **всю папку `paytracker`** (со всеми файлами, включая папки `bot` и `.github`) → **Commit**.

### 3. Узнай chat_id вашего общего чата
- Добавь бота **@payment_ddma_bot** в ваш общий чат (группу).
- Напиши в этом чате **`/start@payment_ddma_bot`** (именно с упоминанием бота — обычное «привет» бот в группе не видит из-за настроек приватности).
- Открой приложение PayFlow → **Настройки** → вставь новый токен → нажми **«🔎 Определить chat_id»**.
- Появится номер чата. У группы он **отрицательный** (например `-1001234567890`). Скопируй его.

### 4. Положи токен и chat_id в секреты GitHub (безопасно)
На странице репозитория: **Settings → Secrets and variables → Actions → New repository secret**. Создай два:
- **BOT_TOKEN** — новый токен из шага 1
- **CHAT_ID** — номер чата из шага 3

### 5. Проверь, что работает
- Вкладка **Actions** → слева **Daily digest** → кнопка **Run workflow** → **Run**.
- Через ~20 секунд в ваш чат придёт дайджест. 🎉

Дальше он будет присылать **сам каждый день в 09:00 МСК** — ничего нажимать не надо.

---

## Как обновлять данные (когда меняются даты оплат)

Бот берёт данные из файла `bot/data.json`. Чтобы он знал свежие даты:
1. В приложении: **Настройки → ⬇ Экспорт** → скачается `payflow-backup.json`.
2. Переименуй его в **`data.json`**.
3. На GitHub зайди в папку **`bot`** → **Add file → Upload files** → загрузи `data.json` (заменит старый) → **Commit**.

Делай это, когда добавила/поменяла оплаты. Каждый день — не нужно.

---

## Поменять время рассылки
Файл `.github/workflows/daily-digest.yml`, строка `cron: '0 6 * * *'`.
Это 06:00 UTC = 09:00 МСК. Чтобы, например, в 10:00 МСК → поставь `0 7 * * *`.

## Важно
- Бот пишет в чат **только если он добавлен в этот чат**.
- Если хочешь личные сообщения каждой вместо общего чата — напиши, переделаю на два получателя.
